# Using the theme
## VARIABLES
The themes uses the following extra vars in `pelicanconf.py`
```
        DESCRIPTION = ''
        TWITTER_URL = ''
        GITHUB_URL = ''
        LINKED_IN_URL = ''
```

## COMPILE
pelican content -t <PATH_TO>/kroustou_pelican

## Upload

        cd output
        git commit -m 'new content'
        gith push
